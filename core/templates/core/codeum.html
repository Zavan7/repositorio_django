Download Automático via CRM

Automação desenvolvida para acessar um CRM, localizar elementos via XPath e realizar downloads automáticos com validação do arquivo.

SOBRE

Projeto criado utilizando Python + Selenium para automatizar o fluxo operacional de downloads dentro de um CRM web.

  • Login automatizado
  • Navegação entre páginas
  • Busca via XPath
  • Download automático
  • Controle de timeout
  • Tratamento de exceções

def download_page(self, timeout=200):

    """
    Realiza download automático via CRM utilizando Selenium.
    """

    try:

        self.driver.find_element(
            By.XPATH,
            '/html/body/form/fieldset/ul/li[11]/button'
        ).click()

        time.sleep(2)

        start_time = start_time()

        while True:

            arquivos = os.listdir(
                self.pasta_download
            )

            if arquivos:

                logging.INFO(
                    f"Arquivo encontrado: {arquivos}"
                )

                break

            elif time.time() - start_time > timeout:

                raise TimeoutError(
                    "Tempo excedido."
                )

            else:
                time.sleep(1)

    except Exception as e:
        logging.error(e)
