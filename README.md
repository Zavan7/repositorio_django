# Portfólio Django

Este é um projeto de portfólio pessoal desenvolvido com Django.

## Status

⚠️ Em desenvolvimento.

## Visão geral

O projeto inclui:

- Aplicação Django com roteamento básico.
- Páginas de portfólio como home, sobre e contato.
- Estrutura de templates e arquivos estáticos.

## Tecnologias

- Python
- Django
- HTML/CSS

## Como executar

1. Crie e ative um ambiente virtual.
2. Instale as dependências.
3. Execute o servidor Django com:

```bash
python manage.py runserver
```
