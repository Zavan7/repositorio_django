# Portfólio Web • Vitor Zavan

Portfólio desenvolvido com **Python + Django**, focado na apresentação de projetos de automação, RPA, backend e desenvolvimento web.

O objetivo principal deste projeto é servir tanto como apresentação profissional quanto como ambiente prático de estudo para aprofundamento em tecnologias web utilizando Python.

## Como Executar

Clone o projeto:

```bash
git clone https://github.com/Zavan7/seu-repositorio.git
```

Entre na pasta:

```bash
cd portifolio_zavan
```

Crie o ambiente virtual com UV:

```bash
uv venv
```

Ative o ambiente virtual:

### Linux

```bash
source .venv/bin/activate
```

### Windows

```bash
.venv\Scripts\activate
```

Instale as dependências:

```bash
uv sync
```

Execute o servidor:

```bash
python manage.py runserver
```
